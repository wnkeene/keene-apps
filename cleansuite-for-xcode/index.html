CleanSuite for Xcode

Reclaim disk space by cleaning Xcode's derived data, caches, device symbols, archives, and more.

Platform: macOS 14.6+

Cleaning Targets

ItemDescriptionArea
DerivedDataClears build data to force rebuildsSafe
Xcode CacheRemoves module, symbol, and stale documentation caches; active doc cache protectedSafe
SwiftPM CacheDeletes Swift Package Manager cachesSafe
Simulator CacheClears simulator temp filesSafe
SwiftUI PreviewsRemoves preview simulator dataSafe
ArchivesSelect individual archives to clean; latest build per app protected; empty date folders swept automaticallyCaution
Device SymbolsSelect individual symbol sets to clean; latest per device model protectedCaution
Simulator RuntimesDeletes simulator OS images; active runtimes protectedCaution
LogsRemoves diagnostic reports and crash logsCaution
Obsolete SimulatorsRemoves simulator devices that can no longer runCaution

Presets

  • Quick Preset — Safe items only (DerivedData, caches, previews). Obsolete Simulators: orphaned devices only.
  • Deep Preset — All Safe items plus Runtimes, Logs, and Obsolete Simulators. Archives and Device Symbols open a detail panel for granular selection. Obsolete Simulators: orphaned devices plus all non-latest runtime devices, protecting the newest runtime per platform.

Safety Features

  • Trash First — All deletions go to Trash; nothing is permanently removed without your action
  • Put Back — Restore individual items or all items at once from Settings
  • Permanently Delete — Remove items from the Recently Cleaned list and Trash only when you're ready
  • Protected Items — Latest archive per app, latest device symbols per device model, and active simulation runtimes are locked and cannot be selected
  • Active Documentation Cache — Current Xcode's documentation cache is always preserved; only older versions are swept
  • Dry Run Mode — Preview what would be cleaned without deleting anything
  • Xcode Running Warning — Alerts you if Xcode is open before cleaning

Reminders & Scheduling

  • Cleaning Reminders — Weekly, monthly, or quarterly notifications
  • Snooze — Postpone reminder by 1 day, 3 days, or 1 week
  • Overdue Alert — Shows days since last clean on app launch

Detail Panels

  • Archives Panel — Tap Archives to browse all builds per app, see sizes, and pick exactly which to clean. Latest build per app shows a lock icon and cannot be selected.
  • Device Symbols Panel — Tap Device Symbols to browse iOS, watchOS, tvOS, visionOS, and macOS symbol sets with sizes. Latest per device model is locked.

Additional Features

  • Disk Space Scanning — Shows reclaimable size for each category before cleaning
  • Free Disk Space Display — Current available disk space shown at a glance
  • Confirmation Sheet — Review all selections and total space to be freed before committing
  • Cleaning Report — Shows exactly what was cleaned and total space freed
  • Open at Login — Optional launch-at-login for automated reminders
  • Donations — Optional tip jar to support development
