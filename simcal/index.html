SimCal

Calendar Companion for iPhone, iPad, and Mac

About

SimCal is a calendar companion that works with the calendars you already use. Browse your month at a glance with color-coded event indicators, view upcoming events by day, week, or month, and create or edit events without leaving the app. SimCal reads from and writes to your existing calendars through Apple's EventKit — your data stays with Apple's Calendar and iCloud systems, never on third-party servers.

Features

  • Month Calendar View — Navigate months with a clean grid showing color-coded dots for days with events
  • Day Events List — Tap any day to see its events with details at a glance
  • Upcoming Events — Browse events filtered by Today, This Week, or This Month
  • Calendar Filtering — View all calendars or filter to a specific one
  • Create & Edit Events — Full event editor with title, dates, all-day toggle, and annual repeat
  • Reminders — Set default reminders with customizable presets for new events
  • Locations — Add locations manually or choose an address from Contacts, then open in Maps
  • Default Calendar — Pick which calendar new events are created in
  • Universal App — Works on iPhone, iPad, and Mac (via Mac Catalyst)

Platform Support

  • iPhone — iOS 18.0+
  • iPad — iPadOS 18.0+
  • Mac — macOS 15.0+ (via Mac Catalyst)
