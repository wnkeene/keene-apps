Kolorz

Native Mac color picker — pick, copy, and convert HEX, RGB, and HSL.

Platform: macOS 14.0+

What It Does

Kolorz is the simplest way to pick a color on macOS and copy it in the format you actually need. It uses Apple’s built-in color picker — the same one you already know from Pages, Keynote, and Preview — and adds the one thing it has always been missing: every color value, ready to copy.

Output Formats

FormatExample
HEX#FF6B35
RGBrgb(255, 107, 53)
HSLhsl(16, 100%, 60%)

Pick

  • Eyedropper — Click any pixel on screen with the system magnifier loupe.
  • Color Wheel, Sliders, Palettes — Every native macOS picking tool, in one window.
  • Image Samplers — Pull palettes from any image you drop in.

Copy

  • One-Click Copy — Click the copy icon on any row to put that format on your clipboard.
  • Drag & Drop — Drag the color preview straight into Sketch, Figma, Xcode, or any app that accepts color drops.

Paste to Convert

  • Bidirectional Conversion — Paste any color value into any row and press Return. Kolorz parses it, applies it to the picker, and updates every other format.
  • Round-Trip Accurate — Works with HEX (including 3-character shorthand like #FFF), RGB, and HSL. No precision loss.

Pin on Top

  • Floating Window — Toggle the pin in the toolbar to keep Kolorz floating above every other app. The picker follows.
  • Persistent — The pinned state is remembered between launches.

Keyboard Shortcuts

  • ⌘1 — Copy HEX
  • ⌘2 — Copy RGB
  • ⌘3 — Copy HSL

Every shortcut is remappable in System Settings ▸ Keyboard ▸ Keyboard Shortcuts ▸ App Shortcuts.

Clear Swatches

To reset the color panel’s swatch grid, choose Color › Clear Swatches… from the menu bar. Kolorz will quit and clear the saved swatches. Relaunch from the Dock or Applications folder to start fresh.

100% Native

  • No Electron — Pure SwiftUI & AppKit. Built on the system color picker.
  • No Tracking — Kolorz does not collect any data. Your colors never leave your Mac.
  • Sandboxed — Runs fully sandboxed on the Mac App Store.
