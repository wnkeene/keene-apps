CleanXcode

Reclaim Disk Space from Xcode on Mac

This app is for private use only. CleanXcode is not available on the Mac App Store and is not intended for public distribution.

About the App

CleanXcode is a private macOS developer utility that scans Xcode-related folders and helps you reclaim disk space safely. Review derived data, caches, old documentation, archives, device symbols, and obsolete simulators from one dashboard — with latest-build protection and everything sent to the Trash first.

Features

  • Reclaimable Summary — Total recoverable space split into Safe and Manage buckets, with last scan time
  • Safe to Clean — One action for DerivedData, stale caches, and old documentation Xcode can rebuild
  • Manage Section — Drill into archives, device symbols, and obsolete simulators with per-item control
  • Latest-Build Protection — Latest archive per app and latest symbols per device model stay locked
  • Trash First — Nothing is permanently deleted until you empty the Trash
  • Xcode-Open Block — Cleaning is refused while Xcode is running
  • Reveal in Finder — Inspect any folder before you remove it
  • Custom Paths — Point the app at non-default DerivedData or Archives locations via sandbox bookmarks

Platform Support

  • Mac — macOS 15.0+
