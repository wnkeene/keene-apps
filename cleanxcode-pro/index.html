CleanXcode Pro

Reclaim disk space by cleaning Xcode's derived data, caches, device symbols, and more.

Platform: macOS 14.6+

Cleaning Targets

ItemDescription
DerivedDataClears build data to force rebuilds
Xcode CacheRemoves module and symbol caches
SwiftPM CacheDeletes Swift Package Manager caches
Simulator CacheClears simulator temp files
SwiftUI PreviewsRemoves preview simulator data
ArchivesDeletes archives (dSYMs will be lost)
Device SymbolsRemoves old debug symbols; latest protected
Simulator RuntimesDeletes simulator OS images; active protected
LogsRemoves diagnostic reports
Obsolete SimulatorsRemoves simulators incompatible with current Xcode

Presets

  • Quick Preset — Safe items only (DerivedData, caches, previews)
  • Deep Preset — All items including archives, symbols, and runtimes

Safety Features

  • Pro Trash — Deleted files go to app's proprietary trash, not system Trash
  • Put Back — Restore individual items or all items from Pro Trash
  • Empty Pro Trash — Permanently delete when you're sure
  • Dry Run Mode — Preview what would be cleaned without deleting
  • Xcode Running Warning — Alert before cleaning if Xcode is open
  • Protected Items — Latest device symbols and active simulator runtimes are never deleted

Reminders & Scheduling

  • Cleaning Reminders — Weekly, monthly, or quarterly notifications
  • Snooze — Postpone reminder by 1 day, 3 days, or 1 week
  • Overdue Alert — Shows days since last clean on app launch

Additional Features

  • Disk Space Scanning — Shows size of each category before cleaning
  • Confirmation Sheet — Review selections before cleaning
  • Cleaning Report — Shows what was cleaned and space freed
  • Donations — Optional tip jar to support development
