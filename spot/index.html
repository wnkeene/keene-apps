Spot Spray Calculator

Agrochemical Spray Calculator for iPhone, iPad, and Mac

About the App

Spot Spray Calculator is a precision tool for agrochemical spray mixing calculations. Calculate exact amounts for tank mixing using either the label rate method or percentage solution method.

Features

  • Imperial and Metric Systems - Toggle between US and metric units
  • Label Rate/Acre - Calculate by rate per acre or hectare
  • Percentage Solution - Calculate by concentration percentage
  • Multiple Units - oz, lb, pt, qt, gal / mL, L, g, kg
  • Precise Calculations - Decimal precision for accurate results
  • Universal App - Works on iPhone, iPad, and Mac (via Mac Catalyst)
  • Clean Interface - Intuitive Apple-style design

How to Use

  1. Select Imperial or Metric unit system
  2. Choose calculation mode: % Solution or Rate/Acre (or Rate/Hectare)
  3. Select unit of measurement for label amount
  4. Enter label rate or percentage
  5. Enter tank size
  6. Tap "=" to see result

Platform Support

  • iPhone - iOS 18.0+
  • iPad - iPadOS 18.0+
  • Mac - macOS 15.0+ (via Mac Catalyst)
